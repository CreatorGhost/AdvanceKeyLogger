Core Features

Real-time Capture

5 built-in capture modules with configurable intervals, deduplication, and smart batching.

Learn more →
Encrypted Transport

AES-256 encryption with circuit breaker pattern, retry logic, and multiple delivery channels.

Learn more →
Plugin System

Registry-based architecture. Write custom capture or transport plugins with a simple decorator API.

Web Dashboard

Built-in FastAPI dashboard with real-time updates, JWT auth, and analytics views.

Privacy Filters

Redaction engine for sensitive data. Application blocklist. Time-based capture rules.
